Description Bitmap Library
**********************************
In the Bitmap Library you will find a set of bitmaps, which can be used 
for the panel design. 
These bitmaps can be used for all bitmap elements of the panel editor. 
The library is divided into two categories:
- Automotive 
- Global 

The categories are each divided into control and display elements with two
and three states. If you like to use bitmaps in your panels, copy
these bitmaps into the directory of the appropriate CANoe configuration. 
Then you can link the elements in the panel editor with the desired bitmap. 

The naming of the bitmaps is made up as follows:

x<Name>_n.bmp

x = size (s/m/l)
n = number of states (n>=2)

for example: sLEDgreen_2.bmp



Beschreibung Bitmap Library
**********************************

In der Bitmap Library finden Sie eine Zusammenstellung von Bitmaps, die für die Panelgestaltung
verwendet werden können. 

Hier enthaltene Bitmaps sind verwendbar für alle Bitmap-Elemente des Paneleditors.
Die Bibliothek ist unterteilt in zwei Rubriken:
- Automotive 
- Global

Die Rubriken sind jeweils unterteilt in Bedien- und Anzeigeelemente mit zwei und drei Zuständen. 
Bitmaps, die Sie in Ihren Panels verwenden möchten, kopieren Sie zuvor in das  Verzeichnis der 
entsprechenden CANoe-Konfiguration. Anschließend können Sie im Paneleditor die Elemente mit
der gewünschten Bitmap verknüpfen.

Die Bezeichnung der Bitmaps setzt sich folgendermaßen zusammen:

x<Name>_n.bmp

x = Größe (s/m/l)
n = Anzahl der Zustände (n>=2)

zum Beispiel: sLEDgreen_2.bmp
